## Loop Solutions

Padrão MVC

controller=orquestrador entre model e view (camada de rotas)  
models=regras de negócio (camada de repositorios)  
view=apresentação visual (templates HTML)

**FERRAMENTAS**  
Python 3.12  
Sqlite3 como banco de dados local  
Fastapi como framework para construção da api  
Uvicorn como servidor ASGI para gerenciar eventos ssíncronos  
Junja2Templates como motor de templates
